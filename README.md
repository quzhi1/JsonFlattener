# JsonFlattener
## Installation

```bash
bin/setup
```

## Usage

```bash
cat test/test_files/test.json | ./bin/json_flattener > temp.json
```

## Testing
In the repository root, run
```bash
ruby -Ilib:test test/json_flattener_test.rb
```
